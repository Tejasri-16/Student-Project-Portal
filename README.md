Student Project Portal:

A centralized, digital hub designed to organize academic projects, track task statuses, and manage deadlines effectively.

Features:
Task Management: Easily track project progress (Pending, In Progress, Completed).

Secure Authentication: Built-in Login and Register functionality to keep your data secure.

Centralized Dashboard: A clean interface to view all your project activities in one place.

Technologies Used:

Frontend: React, HTML, CSS

Backend: Node.js, Express

Database: MongoDB
